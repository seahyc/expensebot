## Curated rules (USER-EDITED — survives schema refresh)

### Custom field glossary
- **Business Trip Destination** → city or country, e.g. "Jakarta", "San Francisco"
- **Business Trip Start / End** → ISO YYYY-MM-DD, the trip window dates
- **Travel - International Sub-Category** → pick one of the schema options. Heuristics:
  - Airline ticket / flight purchase → Flight Ticket (2079)
  - Reschedule fees on flights → Flight Ticket (2079)
  - Hotels, Airbnb during the trip → Accommodations (2078)
  - Rides between airport and home/hotel → Transportation (Airport) (2082)
  - Local rides during the trip → Transportation (2083)
  - Visa application or border tax → Entry Visa and Travel Taxes (2084)
  - Roaming SIM / data package → SIM or Telco Services (2085)
  - Travel insurance policies → Travel Insurance (2086)
- **Travel - Local Sub-Category**:
  - Local Grab/Gojek for work meetings → Transportation (2109)
  - Hotels in SG (rare) → Accommodations (2110)
- **Amount in Payroll Currency** → SGD equivalent. If receipt is already SGD,
  copy claim amount. If foreign currency, prefer the value from user's card
  statement; fall back to fxratesapi at receipt date.

### Classification rules (built up over time)
- Grab/Gojek to/from Changi Airport → Travel-International / Transportation (Airport)
  - Heuristic: pickup or dropoff contains "Changi", "T1", "T2", "T3", "T4", "Airport Boulevard"
- Grab/Gojek otherwise within Singapore → Travel-Local / Transportation
- Cursor / Linear / Notion / GitHub Copilot subscriptions → Subscriptions - Development Tools (3855)
- Anthropic API / Claude subscription → Subscriptions - Development Tools (3855)
- Hotels in JP / ID / MY / TH / VN / etc → Travel-International / Accommodations
- Traveloka, Trip.com, Booking.com flight purchases → Travel-International / Flight Ticket
- For trip-related receipts: when in active trip context, pre-fill trip dates +
  destination from the trip; only override if user mentions different dates

### Description style preferences
- Concise sentence, mention purpose ("for X investor meeting", "client lunch with Y")
- Always include trip name when international (e.g. "Jakarta investor trip")
- For airport rides, include direction ("home → airport" or "airport → home")
